Onboarding Device Instructions

Follow these steps to onboard a new machine to the System Prep Dashboard:

  1. Download the Onboarding Agent: The executable file (`onboarding_agent.exe`) is provided below.
  2. Transfer the File: Copy the downloaded `onboarding_agent.exe` file to the machine you wish to onboard. You can use a USB drive, network share, or any other preferred method.
  3. Run the Agent: On the target machine, simply **double-click** the `onboarding_agent.exe` file.
    Note: You may see a Windows SmartScreen or antivirus warning. You will need to allow the application to run.
  4. Verification: Once the agent runs, it will collect basic system information and send it to the System Prep Dashboard. The machine should then appear in the "Machines Overview" table shortly.
  5. Initial Setup: After the machine appears, you can assign its department, add optional tools, and track its preparation status directly from the dashboard.
